# 🎓 AI-Powered Learning Companion

An interactive AI tutor built with Streamlit that helps students learn Math, Physics, and Chemistry using:

- 💬 Text and voice input
- 🖼️ Image-based question extraction (OCR)
- 📄 File uploads (PDF, DOCX, TXT)
- 🤖 Gemini-powered explanations
- 🧠 Daily Quiz & Weekly Revision

## 📂 Features

- **Multilingual support:** English, Telugu, Hindi
- **Image OCR:** Extracts questions from uploaded images
- **File Uploads:** Reads PDFs, Word Docs, and text files
- **Voice Input:** Uses your mic to convert speech to text
- **Gemini API Integration:** Gets intelligent answers
- **Quiz Module:** Practice MCQs with instant feedback
